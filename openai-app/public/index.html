🎤 SNH48 MCP App

这是一个基于 OpenAI Apps SDK / MCP Apps 标准的远程 MCP 服务器， 提供 SNH48、GNZ48、BEJ48、CKG48、CGT48 成员资料与近期公演票务查询，结果以带成员照片的可视化卡片展示。

MCP 端点：/api/mcp（也可通过 /mcp 访问）

工具：

  • search_members — 按姓名/团体/队伍/期数/籍贯查询成员，展示照片卡片
  • get_upcoming_shows — 查询近期公演场次与票务状态

在 ChatGPT（开发者模式）或 Claude 中添加此连接器时，使用本页面所在域名下的 /api/mcp 作为服务器地址。
